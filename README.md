# Tucil2 IF2211 - Voxelization Objek 3D menggunakan Octree

## Deskripsi
Program ini mengubah model 3D .obj menjadi model voxel berbasis octree dengan pendekatan divide and conquer.

## Requirement
- Go 1.20+

## Cara Menjalankan
Masuk ke folder src lalu jalankan:
go run . ../test/contoh.obj 3

## Struktur Folder
- src: source code
- test: file input/output uji
- doc: laporan
- bin: executable (opsional)
